# AutoHeal — Decision Interface

Interactive digital interface for the AutoHeal Bandage project, built for Minnesota HOSA Medical Innovation (Wayzata High School, Chapter 11029).

## 🚀 Running on GitHub Pages

1. Create a new GitHub repository
2. Upload both files (`index.html` and `README.md`) to the repo
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save** — your site will be live at `https://yourusername.github.io/your-repo-name`

## 🖥️ Running Locally

Just open `index.html` in any browser — no server or install needed.

## Features

- Live sensor readings (temperature, moisture, pH, strain)
- AI chip decision tree showing each autonomous action
- Three simulation modes: Normal Healing, Infection Risk, Swelling
- Adaptive compression, drug release, and sterilization status
- Haptic feedback channel
- ML recovery timeline with projected discharge date
